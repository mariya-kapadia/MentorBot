# AI-Powered Career Mentorship Chatbot (MentorBot)

MentorBot is a responsive, AI-powered web chatbot designed to provide structured guidance on careers, internships, and skill development. The application offers a clean interface with a focus on clarity, usability, and mobile friendliness.
